# New Client Intake Checklist
**Purpose:** Gather baseline information to create a comprehensive client profile.

Use this checklist when onboarding a new client. Collect this info BEFORE running the profile builder.

**Sections match the 13-section Client Profile format for easy data transfer.**

## After Completing This Checklist

1. **Store raw notes** in `04_Discovery_Research/01_Discovery Phase/`
2. **Store credentials** in `02_Onboarding_Access/`
3. **Store brand assets** in `03_Brand_Assets/`
4. **Export any Notion pages** to `notion_export/`
5. **Run the profile builder:**
   ```bash
   ./scripts/run_profile_builder.sh "/path/to/client-xxx"
   ```
6. **Review outputs** and fill any gaps manually
7. **Validate profile** against SOW
