## Hashtag Strategy

### Hashtag Approach
Facebook hashtags are minimal - used for categorization, not discovery.

- **Per post:** 1-3 hashtags maximum
- **Focus:** Branded + 1-2 topical

### Standard Hashtag Set
```
#[BrandedHashtag] #[IndustryTerm] #[LocalHashtag]
```

### When to Skip Hashtags
- Personal/emotional posts
- Link posts (let the link speak)
- Short announcements

## Link Strategy

### Link Post Best Practices
- Short caption (under 100 characters)
- Let preview image/title do heavy lifting
- Strong CTA in caption

### Link Destinations
| Content Type | Link To |
|--------------|---------|
| Blog posts | Website blog |
| Booking/Contact | Scheduling page |
| Products/Services | Service page |
| Resources | Lead magnet / Download |

### Link in Comments (Alternative)
- Post engaging content as image
- Add link in first comment
- Works for algorithm-sensitive content

## Algorithm Considerations (2025)

- **Meaningful interactions:** Comments and shares > reactions
- **Native content preferred:** Uploaded videos > YouTube links
- **Groups get organic reach:** Invest in community
- **Link posts get less reach:** Use strategically, not as default
- **Consistency matters:** Regular posting builds momentum
