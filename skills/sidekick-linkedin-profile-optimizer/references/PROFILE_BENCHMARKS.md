# LinkedIn Profile Benchmarks

## Profile Completeness Checklist

| Element | Status | Impact |
|---------|--------|--------|
| Professional headshot | Required | 14x more profile views |
| Custom headline | Required | First thing people see |
| About section | Required | Tells your story |
| Banner image | Recommended | Brand reinforcement |
| Featured section | Recommended | Showcase best work |
| Experience (current) | Required | Establishes credibility |
| Experience descriptions | Recommended | SEO + depth |
| Skills (top 3 pinned) | Recommended | Endorsement focus |
| Custom URL | Required | Professionalism |
| Contact info | Recommended | Enable connection |
| Creator mode | Optional | For content-focused profiles |

## About Section Best Practices

### Personal Profile Structure
1. **Hook** (2 lines visible before "see more")
2. **Who you help** (audience)
3. **What you do** (services/expertise)
4. **Why you** (credibility/story)
5. **CTA** (what to do next)

### Company Page Structure
1. **What we do** (clear value prop)
2. **Who we serve** (target audience)
3. **Why us** (differentiators)
4. **CTA** (website/contact)

## Benchmark Stats

| Metric | Below Average | Average | Good | Excellent |
|--------|---------------|---------|------|-----------|
| Profile views/week | <50 | 50-100 | 100-300 | 300+ |
| Search appearances/week | <20 | 20-50 | 50-100 | 100+ |
| SSI score | <40 | 40-60 | 60-75 | 75+ |

## SEO Keywords

Include relevant keywords in:
- Headline
- About section (naturally)
- Experience titles and descriptions
- Skills section
- Company specialties

**Tip:** Use keywords your target audience would search for, not internal jargon.
