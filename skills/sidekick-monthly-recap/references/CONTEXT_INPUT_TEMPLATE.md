# Context Input Template

Copy and fill this out (briefly!) when generating a monthly recap. This provides the "Why" behind the "What" (metrics).

## 1. The Narrative Context
*Bullet points are fine.*

**What was the main focus this month?**
- (e.g., "Promoting the Fall Open House", "Driving website traffic", "Brand awareness")

**What were the big wins?**
- (e.g., "Reel X went viral", "Sold out the event", "Lowest CPA on record")

**What challenges did we face?**
- (e.g., "Ad costs rose due to election season", "Website went down for 2 days")

**What are we testing/learning?**
- (e.g., "Tested static images vs carousels - carousels won")

## 2. Next Steps
**What is the focus for next month?**
- (e.g., "Holiday sales push", "New product launch")

**Any specific recommendations for the client?**
- (e.g., "Increase budget for Q4", "Send us more raw video footage")
